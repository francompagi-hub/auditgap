The Assessment

What the audit covers

Our WCAG 2.1 AA gap analysis combines automated scanning with structured manual review across five critical areas. Every finding is verified, documented, and delivered with remediation guidance.

  • Automated WCAG 2.1 AA scanning — all pages

    Full-site scan using industry-standard accessibility testing tools. Every page indexed and assessed against WCAG 2.1 Level AA criteria. Not just the homepage.

  • Manual keyboard navigation testing

    Automated scans miss 30–40% of real accessibility barriers. Manual testing verifies that all interactive elements — menus, dropdowns, cart, checkout — are reachable and operable without a mouse.

  • Screen reader compatibility review

    Testing for ARIA labels, alt text, heading structure, and focus management. Identifies the specific barriers that prevent screen reader users from completing a purchase.

  • Color contrast and visual accessibility checks

    All foreground/background combinations measured against WCAG 4.5:1 contrast ratio requirements. Typography size, weight, and readability evaluated.

  • Form, checkout, and cart functionality audit

    The highest-risk area for e-commerce stores. Error messaging, label associations, submit button accessibility, and payment form compliance all assessed.
